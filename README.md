#  Customer Churn Analysis (SQL)

## Overview
This project analyzes customer churn behavior using SQL queries.  
It focuses on identifying patterns in customer retention and understanding factors influencing churn.

---

##  Key Objectives
- Calculate total customers and churn rate  
- Analyze churn across different plans  
- Identify high-risk customers  
- Compare average charges between churned and retained customers  

---

##  Tools Used
- SQL Server (SSMS)

---

##  Key Queries
- Total customers & churn rate  
- Plan-wise churn analysis  
- Gender-wise churn analysis  
- Average monthly charges comparison  
- Top paying customers  

---

##  Key Insights
- Basic plan customers show higher churn  
- Low tenure customers are more likely to churn  
- Premium users show better retention  
- Higher charges may influence churn behavior
